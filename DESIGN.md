# Design System

This document outlines the core visual principles and design tokens for our product, ensuring consistency and a cohesive user experience.

## Theme Overview

Our design system utilizes a **light color mode**, providing a bright and airy aesthetic. The overall design emphasizes clarity and user-friendliness.

## Typography

### Headlines
Headlines use the **Inter** font family, chosen for its modern feel and excellent readability across various sizes.

### Body Text
Body text also employs the **Inter** font family, ensuring a consistent typographic voice throughout the UI and optimal legibility for longer content.

### Labels
Interactive elements and labels feature the **Inter** font family, providing a clean and direct visual cue for user actions.

## Color Palette

The color palette is built around a harmonized set of colors, ensuring accessibility and brand recognition.

*   **Primary Color:** `#6750a4` - A deep, rich purple serving as the brand's most distinctive chromatic color, ideal for call-to-action buttons and key interactive elements.
*   **Secondary Color:** `#625b71` - A sophisticated, muted purple, supporting less prominent UI elements, chips, and secondary actions.
*   **Tertiary Color:** `#7d5260` - A warm, earthy rose tone, providing an additional accent for highlights, badges, or decorative elements.
*   **Neutral Color:** `#79747e` - A versatile, balanced gray, forming the base for backgrounds, surfaces, and non-chromatic UI components.

## Shape and Form

### Roundedness
UI elements feature **moderate roundedness (2)**, striking a balance between modern aesthetics and approachability, avoiding overly sharp or extremely soft edges.

## Layout and Spacing

### Spacing
The design system adopts a **normal (2)** level of spacing, providing adequate breathing room between UI elements without feeling overly sparse or dense. This ensures a comfortable and organized visual presentation.
